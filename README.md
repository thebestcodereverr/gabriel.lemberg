# Resume Website

This workspace contains a simple static resume website that you can personalize and deploy.

## 🧰 What's Included

- `index.html` — your resume content (replace the placeholders with your own info)
- `styles.css` — clean responsive styling that works on mobile and desktop

## 🚀 How to Run Locally

1. Open this folder in your code editor (e.g., VS Code).
2. Open `index.html` in your browser. (You can just double-click it.)

If you want a local server (recommended):

```bash
# Using Python 3
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## ✨ Customize

- Update your name, title, and contact info inside `index.html`.
- Add more experience/projects by copying the existing `<div class="card">` blocks.
- Add Word/PowerPoint/PDF files into the repo and link them from a project card (see the “Presentation / Report” card in `index.html`).
- Update colors or spacing in `styles.css`.

## 📦 Deploy Options

- GitHub Pages: push this repo and enable Pages in the repo settings.
- Netlify / Vercel: connect the repo and deploy.
